# ①課題名
ガマン貯金アプリ（我慢した分を可視化する節約記録アプリ）

## ②課題内容（どんな作品か）
衝動的にお金を使いそうになったときに「買わなかったもの」と「金額」を記録し、我慢できた分の合計金額を月単位で可視化するアプリです。

節約を「我慢」や「反省」ではなく、自分を褒める行為として記録できることを目的にしています。

主な機能は以下です。
・我慢した内容の登録（確認画面あり）
・登録内容の編集・削除
・今月どれくらい節約できたかの合計表示
・一覧での履歴確認

日常生活で実際に使えることを意識して制作しました。
